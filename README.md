# Intruder_Log
Python based tool for spotting suspicious SSH login attempts.

**What I learned from building this tool:**

- variables
- Reassigining and updating values (count +=1)
- Data types: strings, integers, floots and booleans
- Nested dictionaries
- Lists
- Indexing
- Slicing
- String methods
- Functions
- parameters and arguments
- return, if, else, in, not in
- "for loops"
- f-strings
- format specifiers (percentage with 2 decimals)
- Math operators * / + 
- Built in functions (open, input(), print(), sorted() )
- Key arguments (key=, reverse=True)
- Reading files( open, .readlines(), .close )
- External libraries (import, pip. pygiglet)
- ANSI color codes
- Program logic

## Disclaimer 
This tool is made as fun-project to learn python-coding.
This is not intended for production use, and is not be treated as replacement for any kind of security software.
